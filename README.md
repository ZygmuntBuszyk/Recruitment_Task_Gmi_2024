# Task Management Application

A full-stack task management application built with React Native (Expo) and NestJS.

## Project Structure

- `/frontend` - React Native mobile application
- `/backend` - NestJS API server

## Quick Start

### Prerequisites
- Node.js
- Docker & Docker Compose
- npm or yarn
- Expo CLI

### Backend Setup
```bash
cd backend
npm install
docker-compose up -d
npm run start:dev
